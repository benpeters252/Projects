Title: Minneapolis Housing Price Prediction

Data Source: https://www.kaggle.com/datasets/thuynyle/redfin-housing-market-data


Problem Statement: In order for the homebuyer to have a better idea of housing costs, a model could be used to predict the cost, and then the home buyer could compare to the expected price given by the model.

Goal: Build a Model that can be used to determine if a property is over/under valued
